# Project Structure

## Pastas Principais

- `assets/`: todos os recursos estáticos.
- `src/`: código-fonte (CSS, JS e dados).
- `docs/`: documentação do projeto.

## Árvore Resumida

```text
.
|-- assets
|   |-- animations
|   |-- images
|   `-- videos
|-- docs
|   `-- PROJECT-STRUCTURE.md
|-- src
|   |-- css
|   |-- data
|   `-- js
|-- index.html
`-- README.md
```

## Padrão de Nomenclatura

- Arquivos em `kebab-case` (ex.: `quantum-battle-system.js`).
- Nomes de pastas em minúsculo e sem abreviações ambíguas.

## Regras de Organização

- Mídia não deve ficar misturada com código.
- Novas folhas de estilo devem entrar em `src/css`.
- Novos scripts devem entrar em `src/js`.
- Novos datasets devem entrar em `src/data`.
